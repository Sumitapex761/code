Jenkins + SCM Integration

Step-by-Step Setup

  1. Open Jenkins Dashboard
  2. Create New Item → Enter name → Select "Freestyle project"
  3. Go to Source Code Management section
  4. Select Git
  5. Enter Repository URL (e.g., https://github.com/username/repo.git)
  6. Optional: Add credentials if it's private
  7. Set Branch (e.g., main or master)
  8. Save the project

Trigger Build Automatically

Go to Build Triggers and check Poll SCM

Enter schedule: H/5 * * * * (every 5 mins)

Build Output

Jenkins will pull code from Git and run the build steps you define.
